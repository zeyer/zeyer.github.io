Hexo+GitHub搭建个人博客

发表于 2016-05-04   |   分类于 教程

说明

本教程采用Github + Hexo的方式搭建博客，优点是免费。Hexo版本为3.1.1，采用第三方主题nexT。总体流程如下

  1. 安装node.js 和 Git
  2. 申请Github账号，并开启github免费blog
  3. 本地搭建Hexo 的blog站点，发布到Github上

本教程参考如下网站

http://www.jianshu.com/p/985d07d88ef4
